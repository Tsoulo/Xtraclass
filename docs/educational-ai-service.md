# Educational AI Service Documentation

## Overview
The Educational AI Service is a comprehensive AI integration system that provides automated grading, personalized feedback, and adaptive exercise generation for the XtraClass.ai platform. It uses OpenAI's GPT-3.5-turbo model for cost-efficient educational AI operations.

## Core Terms and Concepts

### Educational Context
A data structure that provides curriculum and learning context to the AI system:

```typescript
interface EducationalContext {
  grade: string;           // Student grade level (e.g., "8", "9", "10")
  subject: string;         // Subject area (e.g., "mathematics", "science")
  topic: string;          // Current topic (e.g., "Algebra", "Geometry")
  theme?: string;         // Specific theme within topic
  difficulty: 'easy' | 'medium' | 'hard';  // Exercise difficulty
  syllabus: 'CAPS' | 'IEB';               // Curriculum standard
  term?: string;          // Academic term
  week?: string;          // Week number
}
```

### Generated Exercise
AI-created educational content with structured questions:

```typescript
interface GeneratedExercise {
  id: string;             // Unique identifier
  title: string;          // Exercise title
  description: string;    // Exercise description
  questions: Question[];  // Array of questions
  totalMarks: number;     // Total possible points
  estimatedDuration: number; // Time in minutes
  difficulty: 'easy' | 'medium' | 'hard';
}
```

### Question Structure
Individual question within an exercise:

```typescript
interface Question {
  id: string;             // Question identifier
  question: string;       // Question text
  answer: string;         // Correct answer
  marks: number;          // Points value
  type: 'short-answer' | 'multiple-choice' | 'essay';
}
```

### AI Feedback
Comprehensive feedback generated by the AI system:

```typescript
interface AIFeedback {
  overall: {
    score: number;        // Points earned
    percentage: number;   // Percentage score
    grade: string;        // Letter grade
    strengths: string[];  // Student strengths
    improvements: string[]; // Areas for improvement
  };
  questionFeedback: QuestionFeedback[];
}
```

## Core Functions

### 1. Exercise Generation

#### `generateExercise(context: EducationalContext, numQuestions: number = 5)`
Generates new educational exercises based on curriculum context.

**Parameters:**
- `context`: Educational context defining curriculum requirements
- `numQuestions`: Number of questions to generate (default: 5)

**Returns:** `GeneratedExercise`

**Usage:**
```typescript
const exercise = await educationalAIService.generateExercise({
  grade: "8",
  subject: "mathematics",
  topic: "Algebra",
  difficulty: "medium",
  syllabus: "CAPS"
}, 5);
```

#### `generateAdaptiveExercise(context: EducationalContext, feedbackContext: FeedbackContext, numQuestions: number = 5)`
Generates personalized exercises targeting specific student weaknesses.

**Parameters:**
- `context`: Educational context
- `feedbackContext`: Student performance data and improvement areas
- `numQuestions`: Number of questions

**Features:**
- Targets specific student mistakes
- Includes exact improvement areas from previous feedback
- Aligns with CAPS curriculum requirements
- Adjusts difficulty based on performance

### 2. Feedback Generation

#### `generateFeedback(exercise: GeneratedExercise, studentAnswers: string[], context: EducationalContext)`
Analyzes student submissions and provides detailed feedback.

**Parameters:**
- `exercise`: The exercise being graded
- `studentAnswers`: Student's submitted answers
- `context`: Educational context

**Returns:** `AIFeedback`

**Features:**
- Comprehensive scoring with percentage and letter grades
- Identifies specific strengths and improvement areas
- Per-question detailed feedback
- Mathematical answer normalization

### 3. Homework Generation

#### `generateHomework(context: EducationalContext, requirements: HomeworkRequirements)`
Creates structured homework assignments with multiple questions.

**Parameters:**
- `context`: Educational context
- `requirements`: Homework specifications (duration, difficulty, etc.)

**Returns:** `GeneratedHomework`

**Features:**
- Multi-question homework structure
- Estimated completion time
- Total points calculation
- CAPS curriculum alignment

### 4. Service Management

#### `getStatus()`
Returns current service configuration and health status.

**Returns:**
```typescript
{
  model: string;           // AI model being used
  apiConfigured: boolean;  // API key status
  message: string;         // Status message
}
```

## API Endpoints

### Exercise Generation
- `POST /api/educational-ai/generate-exercise` - Generate new exercise
- `POST /api/generate-adaptive-exercise` - Generate adaptive exercise

### Feedback and Grading
- `POST /api/educational-ai/generate-feedback` - Generate feedback
- `POST /api/mock-grading` - Grade student submissions

### Service Status
- `GET /api/educational-ai/status` - Check service health

### Testing
- `GET /api/test-ai` - AI service testing interface

## Integration Examples

### Basic Exercise Generation
```typescript
// Generate a mathematics exercise
const exercise = await educationalAIService.generateExercise({
  grade: "8",
  subject: "mathematics", 
  topic: "Linear Equations",
  difficulty: "medium",
  syllabus: "CAPS"
});
```

### Adaptive Exercise with Feedback Context
```typescript
// Generate exercise targeting specific weaknesses
const adaptiveExercise = await educationalAIService.generateAdaptiveExercise({
  grade: "8",
  subject: "mathematics",
  topic: "Algebra", 
  difficulty: "medium",
  syllabus: "CAPS"
}, {
  previousPerformance: 45,
  weakAreas: ["equation solving", "variable isolation"],
  specificMistakes: ["forgot to add 7 to both sides", "division error"]
});
```

